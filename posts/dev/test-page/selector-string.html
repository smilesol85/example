선택자[속성~=값]

속성 안의 값이 특정 값을 단어로 포함하고 있는 것 선택

class : strSlt-a

data-role="ab abc"

[data-role~=ab]{color:red;}

선택자[속성|=값]

속성 안의 값이 특정 값을 단어로 포함하고 있는 것 선택

class : strSlt-b

data-role="ko-kr"

[data-role|=ko]{color:red;}

선택자[속성*=값]

속성 안의 값이 특정 값을 포함하고 있는 것 선택

class : strSlt-b2

data-role="abxdfsdfwefsdf"

[data-role*=x]{color:red;}

선택자[속성^=값]

속성 안의 값이 특정 값으로 시작하는 태그를 선택

class : strSlt-c

data-role="abcdef"

[data-role^=a]{color:red;}

선택자[속성$=값]

속성 안의 값이 특정 값으로 끝나는 태그를 선택

class : strSlt-d

data-role="abcdef"

[data-role$=f]{color:red;}
