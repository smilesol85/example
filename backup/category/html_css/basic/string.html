    홍길동은 blah blah~알고 했습니다.
    <q> 인용구는 따옴표(" ")가 함께 표시된다.

  • 글자 형태

    [b] 굵은 글자 태그
    [i] 기울어진 글자 태그
    [small] 작은 글자 태그
    test[sub] 아래에 붙는 글자
    test[sup] 위에 붙는 글자
    [ins] 밑줄 글자 태그
    [del] 가운데 줄이 그어진 글자 태그
